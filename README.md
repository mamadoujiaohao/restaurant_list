# restaurant_list
<h1>這是一個用node.js、express、handlebars、架設的餐廳清單網站</h1>
<h3>下載檔案後,請於根目錄使用 npm install載入需要的模組</h3>
<h3>目前只有8家餐廳,但相信日後會越來越多</h3>
<h3>在網頁主頁的每一間餐廳,都可以點進去看其詳細的介紹</h3>
<h3>上方的搜尋欄可快速搜尋您想知道的餐廳(目前只有8間)</h3>
